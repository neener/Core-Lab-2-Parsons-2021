Core 1 Lab Skills Test : Files and Folders

The goal of this exercise is to share how you organize your files and folders while testing your HTML & CSS skills from Core Lab 1.

Guide:

Use phrases in bold to reflect on files, folders, and organization

Use phrases in code to refresh html & css skills

You can use either glitch or a text editor to complete this assignment.

★HTML review★

□ Make a list of what you use to take notes using an unordered HTML list and list items.

□ Using an h3 tag give your list a title.

□ Insert an image describing how you archive your digital photos. Don't forget to give the image an alt description.

□ Using a blockquote element describe your image and align the text to the right of the image.

□ Using a table element list the differences between organizing digitally and organizing physically.

□ Create a list of links of external services you use to stay organized.

★CSS review★

□ Change the font-family you are using in your h3 tag

□ Duplicate the image from above and alter it, ie. warp, crop, resize

□ Describe your most important files, use this text to wrap around the image you duplicated.

□ Change the border color of the table, fill the first table row with a color, the first column with a different color, and the second column with another color.

□ Add a hover effect on your list of links to change either the color, size, or shape of text when hovered.

□ Add a background color or colors to your page and change the opacity.

★Bonus:

□ Upload your project to Github and share your link in Notion.

□feel free to go wild and show off other html,css skills :)
